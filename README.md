## ⚙️ Campaign Setup — How I Run Campaigns

### Step 1: Lead Sourcing
```
Tools Used: Apollo.io · LinkedIn Sales Navigator · Hunter.io
- Filter by: Industry, Job Title, Company Size, Location
- Verify emails before upload (reduce bounce rate)
- Export clean CSV list
```

### Step 2: Email Account Warmup
```
Tools Used: Instantly · FoxReach
- Minimum 2-3 weeks warmup per account
- Use multiple accounts to distribute volume
- Monitor spam scores weekly
```

### Step 3: Sequence Writing
```
Step 1 — Cold Email (Day 1)
  Subject: Short, curiosity-driven, personalized
  Body: Problem → Solution → CTA (3-4 lines max)

Step 2 — Follow-up (Day 4-5)
  Subject: Re: [original thread]
  Body: Soft bump, add value, different angle
```

### Step 4: Campaign Configuration
```
Platform      : FoxReach
Sending Window: US timezone aligned (5PM–11PM PKT)
Active Days   : Mon–Fri (Weekdays)
Daily Limit   : 2,200–2,900 emails/day
Tracking      : Opens, Clicks, Replies, Bounces
```

### Step 5: Monitoring & Optimization
```
- Check bounce rate daily (keep under 5% ideally)
- Pause accounts with high bounce
- A/B test subject lines
- Reply handling within 2 hours
```
